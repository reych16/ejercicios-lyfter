#Big O Algorithm Analysis

This repo contains the analysis of several algorithms using Big O notation.

---

## 1. Bubble Sort
Bubble sort uses two nested loops to compare adjacent elements and swap them if necessary.

**Complexity**
O(n^2)

---
## 2. print_numbers_times_2
The loop iterates once for each element in the list.

**Complexity**
O(n)

---
## 3. check_if_lists_have_an_equal
Two nested loops compare every element in list_a with every element in list_b.

**Complexity**
O(n^2)

---
## 4. print_10_or_less_elements
The loop executes at most 10 times regardless of the list size.

**Complexity**
O(1)

---
## 5. generate_list
Three nested loops generate combinations of three lists.

**Complexity**
O(n^3)
